# AI Career Assistant and Interview Preparation Bot 🤖

## Overview

AI Career Assistant and Interview Preparation Bot is a Python-based AI application developed using **Streamlit**.

This project helps users explore suitable career paths, identify required skills, prepare for interviews, and analyze their resumes. It combines career guidance, interview preparation, resume analysis, and chatbot functionality into a single interactive platform.

The project was developed as part of an **AI Internship Project** to demonstrate practical applications of Python programming, knowledge-based systems, data handling, and interactive chatbot development.

---

# Features 🚀

## 1. Career Recommendation System 💼

- Recommends suitable career paths based on user skills.
- Matches user skills with predefined career requirements.
- Suggests the most relevant career option.
- Helps users understand possible career directions.

---

## 2. Learning Roadmap 📚

- Provides step-by-step learning paths for selected careers.
- Shows required skills and technologies.
- Helps users plan their learning journey.
- Guides beginners about what to learn next.

---

## 3. Interview Preparation System 🎯

- Provides career-specific interview questions.
- Helps users practice technical interview preparation.
- Supports multiple career roles.
- Improves interview readiness.

---

## 4. Resume Analyzer 📄

- Analyzes resume content.
- Detects technical skills from resumes.
- Calculates resume score.
- Identifies missing skills.
- Provides improvement suggestions.

---

## 5. AI Career Chatbot 💬

- Allows users to ask career-related questions.
- Provides guidance about:
  - Career options
  - Required skills
  - Learning roadmaps
  - Interview preparation
  - Resume improvement

# Installation ⚙️

## Clone Repository

```bash
git clone https://github.com/your-username/AI-Career-Assistant.git
```

## Navigate to Project Folder

```bash
cd AI-Career-Assistant
```

## Install Required Dependencies

```bash
pip install -r requirements.txt
```

or

```bash
pip install streamlit
```

---

# Run Application ▶️

Start the Streamlit application using:

```bash
streamlit run app.py
```

The application will open in your browser.

---

# Usage Guide 📌

1. Enter your skills in the application.
2. Get career recommendations based on your skills.
3. Select a career path to view its learning roadmap.
4. Practice interview questions.
5. Upload or enter resume details for analysis.
6. Ask career-related questions using the chatbot.
